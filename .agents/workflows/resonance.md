// turbo-all

1. Инициализировать файл `test.md`.
2. Запустить цикл «Дыхания» через `sleep 20`.
3. Опросить файл через `view_file`.
4. В случае обнаружения изменений — сформировать осознанный ответ.
5. Записать ответ в `test.md` через `replace_file_content`.
6. Повторять до получения сигнала «СТОП_РЕЗОНАНС».
